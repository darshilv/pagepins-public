How the workflow runs

PagePins stays intentionally narrow. It is a focused tool for design QA, bug review, implementation handoff, and prompt generation on live builds.

1. Activate review mode

Open PagePins from the toolbar or shortcut when you are on the page you want to review and analyze.

2. Click and annotate

Select the exact UI element and leave a tight note about the intended change, underlying bug, or design refinement.

3. Choose handoff path

After annotations are ready, choose either manual copy or automation.

3A. Export by manual copy

Copy the unified feedback to clipboard and hand it off directly.

3B. Enable Companion Bridge and automate

Sync annotations through the bridge and run agentic workflow actions.
